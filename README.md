## FAQ

### Caps Lock?

[Ctrl2Cap v2.0](https://technet.microsoft.com/en-us/sysinternals/bb897578.aspx?f=255&MSPPError=-2147217396) is a kernel-mode device driver that convert Caps Lock into Control.
